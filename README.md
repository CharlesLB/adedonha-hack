# Adedonha Hack
Um programa PHP para trapacear na adedonha. Basta dizer as categorias da competição e a letra que a aplicação lhe dará todas as respostas

## Tecnologias

1. PHP
2. Composer
    - CoffeCode\DataLayer
    - CoffeCode\Routes
    - League\Plates
3. AJAX
